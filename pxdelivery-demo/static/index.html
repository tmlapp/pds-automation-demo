About Us

Portworx Delivery is a fake application used to explain how to use containerized application that contain stateful resources such as message queues and databases. This application currently uses MongoDB, Kafka, and MySQL for some examples.

  • Portworx Data Services

    Portworx Data Services is used to quickly spin up data services in Kubernetes

  • Portworx Enterprise

    Portworx Enterprise is used in this application as a way to protect the data services in Kubernetes to ensure data is replicated across nodes among other capabilities.

  • Portworx Backup

    Portworx Backup can also be used for data protection of our Kubernetes Data Services
